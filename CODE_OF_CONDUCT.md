# Code of Conduct

## Our pledge

We are committed to providing a welcoming and inclusive environment for everyone who participates in this project.

## Expected behavior

- Be respectful and professional.
- Use constructive language.
- Be open to feedback.
- Focus on the issue, not the person.
- Respect different perspectives and experience levels.

## Unacceptable behavior

- Harassment or discrimination.
- Personal attacks.
- Trolling or insulting comments.
- Publishing private information without permission.
- Any behavior that disrupts a healthy community.

## Enforcement

Project maintainers may remove, edit, or reject contributions that violate this code of conduct. Repeated or serious violations may result in temporary or permanent removal from the project.

## Reporting

If you experience or witness behavior that violates this code of conduct, report it to the project maintainers through the repository’s preferred private or public support channel.

## Scope

This code of conduct applies to all project spaces, including issues, pull requests, discussions, and related community interactions.
